# Pomodoro Timer for VS Code

Таймер для техники Pomodoro в статус-баре Visual Studio Code по схеме _4 блока по (25 минут работы + 5 минут отдыха), затем 30-минутный перерыв и заново._

## Команды
- **`Pomodoro: начать рабочий процесс`** - запустить таймер
- **`Pomodoro: поставить на паузу / продолжить`** - пауза/продолжение (или по клику на статус-бар)
- **`Pomodoro: закончить рабочий процесс`** - остановить и убрать таймер из статус-бара, посмотреть статистику по проделанной работе

## Как использовать

1. Нажмите `Ctrl+Shift+P` и выберите `Pomodoro: начать рабочий процесс`
2. Работайте 25 минут - таймер покажет оставшееся время в статус-баре
3. Когда время выйдет - автоматически начнется перерыв
4. Для паузы кликните на таймер в статус-баре или используйте команду
5. По завершении работы остановите таймер чтобы увидеть статистику

## Содержание `extension.ts`
- `workDuration, shortBreakDuration, longBreakDuration` - временные интервалы для отдыха, короткого отдыха и длинного отдыха *в секундах*. Можно кастомизировать.
- `updateStatusBar()` - функция обновления статус-бара: формирование надписи и непосредственно обновление с помощью `statusBarItem.show()`.
- `startTimer()` - функция запуска таймера и последующей работы с временем.
- `togglePause() | 'pomodoro.togglePause'` - функция активации паузы / возобновления и соответствующая команда.
- `startCommand | 'pomodoro.startPomodoro'` - команда для запуска расширения: зануление рабочих переменных и выполнение `startTimer()`.
- `deactivateCommand | 'pomodoro.deactivatePomodoro'` - команда для выхода из расширения: скрытие из статус-бара и вывод в уведомлении количества выполненных рабочих интервалов.
