# Pomodoro Timer for VS Code

Таймер для техники Pomodoro в статус-баре Visual Studio Code по схеме _4 блока по (25 минут работы + 5 минут отдыха), затем 30-минутный перерыв и заново._

## Команды
- **`Pomodoro: начать рабочий процесс`** - запустить таймер
- **`Pomodoro: поставить на паузу / продолжить`** - пауза/продолжение (или по клику на статус-бар)
- **`Pomodoro: закончить рабочий процесс`** - остановить и убрать таймер из статус-бара, посмотреть статистику по проделанной работе

## Как использовать

1. Нажмите `Ctrl+Shift+P` и выберите `Pomodoro: начать рабочий процесс`
2. Работайте 25 минут - таймер покажет оставшееся время в статус-баре
3. Когда время выйдет - автоматически начнется перерыв
4. Для паузы кликните на таймер в статус-баре или используйте команду
5. По завершении работы остановите таймер чтобы увидеть статистику

## Описание кода 'extension.ts'
### 1. workDuration, shortBreakDuration, longBreakDuration
```
Временные интервалы для отдыха, короткого отдыха и длинного отдыха в секундах. Можно кастомизировать.
```
### 2. updateStatusBar():
```
Функция обновления статус-бара: 
+ Формирование надписи для статус-бара.
+ Обновление с помощью VS Code API с помощью statusBarItem.show()
```
### 3. startTimer()
```
Функция запуска таймера и обработки времени:
+ Ежесекундное вычисление текущего временного интервала и перехода к следующему.
+ Обновление статус-бара с помощью updateStatusBar().
```
### 4. togglePause() | 'pomodoro.togglePause'
```
Функция активации паузы / возобновления:
+ Инвертирование isPaused
+ Обновление статус-бара с помощью updateStatusBar().
```
### 5. startCommand | 'pomodoro.startPomodoro'
```
Команда: запуск расширения
+ Зануление и установка стандартных значений рабочим переменым
+ Вывод уведомления о запуске
```
### 6. deactivateCommand | 'pomodoro.deactivatePomodoro'
```
Команда: выход из расширения
+ Зануление таймера
+ Скрытие статус-бара
+ Вывод завершающего уведомления
```
